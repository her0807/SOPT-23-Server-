# 5차 세미나 정리

**AWS S3**

 아마존에서 객체 스토리지 서비스를 별도로 제공함

데이터를 bucket 이라는 리소스 내에 저장한다. 

free-tier 는 bucket 에 상관없이 총 5GB, 1달에 2000번 업로드 제공

**EC2** 는 서버 애플리케이션, RDS는 DB ,S3는 멀티 미디어 대용량 저장소이다.

s3만 접근 가능한 IAM 계정을 생성할 때 키 두개를 주는데 **절대유출하면안됩니다ㅠㅠ**
